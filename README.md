## Cara Instalasi
1. git clone origin 
2. composer install
3. php artisan db:seed
4. Jalankan dengan perintah 'php artisan serve'
5. Login Akun : - Admin : superadmin@mail.com (rahasia) - User : user@mail.com (rahasia)
